Paradise Nursery is a dynamic shopping cart web application built using **HTML, CSS, JavaScript, and Node.js (Express)**.

## ğŸ“Œ Project Overview

This project simulates an e-commerce experience for an online plant store. Users can:

- View a landing page with a background image
- Browse plants grouped into categories
- Add plants to a shopping cart
- Increase or decrease item quantities
- Remove items from the cart
- View total price calculations
- See a dynamically updated cart counter
- Navigate using a persistent navigation bar

The cart data is stored using **localStorage**, allowing items to remain in the cart during the session.
